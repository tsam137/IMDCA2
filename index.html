    4th Year Project


    weLearn


    I am designing a web app called GURU, where users can learn languages and teach languages to others that desire to learn them. For instance, if a user already knew English and wanted to learn German, they would look through profiles on the website and find a user that wanted to learn English and already knew German. Users would be able to create an account and add information about the language(s) they already know and the languages they desire to know along with other information like age and country location. Once a profile has been created then a user would be able to send messages to other users to set up times and dates to have meetings online to teach each other the desired languages.

    Internship Experience Project


    GURU - Grow, Understand, Ramp Up


    During my time at SAP, I took part in a project with some other interns in my field. We had various meetings at the beginning to iron out how we would go about doing this project. In those meetings, we decided that we would be fixing a tool in the company that is used for the onboarding of new hires. We named the project GURU which stands for Grow, Understand, Ramp Up. We established that the best methodology to use would be the Design Thinking Methodology. This is a methodology that heavily involves the users that will be using the product. We used programs like Adobe XD to design Low to High Fidelity Prototypes in order to get an idea of what we had to design. We then used Vue.js to code the website for the new Onboarding Tool. This tool had the potential to be used by SAP interns all over the globe as the tool that was used globaly at that time was outdated.
